# WatchX-II

#### 基于AT32F403A的第二代智能手表，图形处理基于LVGL.
>[硬件和UI设计](https://github.com/Trigger-CN/WatchX-II_Hardware) by [Trigger-CN](https://github.com/Trigger-CN)
## Features特性
- 240X240分辨率的触摸屏
- 华丽自然的过度动画
- 美观的UI界面
- 3D打印的手表外壳，厚度仅为1CM
- 动画时间轴功能，让实现华丽复杂的动画成为可能


## Functions功能
- 动态表盘
- 表达式计算器
- 秒表
- 计步
- 基于Aduiboy的游戏机
- 时间日期设置
- 背光设置

## Pages页面
- Page_DialPlate 表盘页面
- Page_Calculator 计算器
- Page_HreatRate 心率
- Page_Game2048 游戏2048
- Page_Game 基于Aduiboy的游戏机
- Paeg_MainMenu 主菜单
- Page_Music 音乐
- Page_Setting 设置
- Page_Sport 运动
- Page_Sleep 睡眠
- Page_Stopwatch 秒表
- Page_TimeSet 时间设置
- Page_BackLightSet 背光设置

## Interaction交互
- 所有交互功能通过电源按键和触摸屏完成
- 点按电源键可完成开机和退出页面，长按电源键为关机，触摸屏具有点按和滑动手势，可供进入和退出页面
